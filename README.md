# LangGraph Beginner Workspace

This workspace contains a series of Jupyter notebooks designed to help you learn and experiment with LangGraph, a framework for building language model applications using graphs.

## Contents

- **1_Hello_World.ipynb**: Introduction to LangGraph with a simple Hello World example.
- **1e_Personalized_Compliment_Agent.ipynb**: Build a personalized compliment agent using LangGraph.
- **2_Multiple_Inputs.ipynb**: Learn how to handle multiple inputs in LangGraph workflows.
- **2e_Plus_Multiply.ipynb**: Explore combining operations (addition and multiplication) in a graph.
- **3_Sequential_Graph.ipynb**: Understand sequential execution of nodes in LangGraph.
- **3e_About_Me.ipynb**: Create an "About Me" agent using LangGraph.

## Getting Started

1. Open the notebooks in your preferred Jupyter environment (VS Code, Jupyter Lab, etc.).
2. Follow the instructions and code cells in each notebook to learn LangGraph step by step.

## Purpose

This folder is intended for beginners who want to get hands-on experience with LangGraph by working through practical examples.
