# LangGraph Beginner Workspace

This workspace contains a series of Jupyter notebooks designed to help you learn and experiment with LangGraph, a framework for building language model applications using graphs.

## Contents

- **1_Hello_World.ipynb**: Introduction to LangGraph with a simple Hello World example.
- **1e_Personalized_Compliment_Agent.ipynb**: Build a personalized compliment agent using LangGraph.
- **2_Multiple_Inputs.ipynb**: Learn how to handle multiple inputs in LangGraph workflows.
- **2e_Plus_Multiply.ipynb**: Explore combining operations (addition and multiplication) in a graph.
- **3_Sequential_Graph.ipynb**: Understand sequential execution of nodes in LangGraph.
- **3e_About_Me.ipynb**: Create an "About Me" agent using LangGraph.
- **4_Conditional_Graph.ipynb**: Learn how to add conditional logic (if/else) to your LangGraph workflows.
- **4e_Two_Routers.ipynb**: Experiment with multiple routers and more complex conditional flows.
- **5_Looping_Graph.ipynb**: Implement looping and iteration in LangGraph graphs.
- **5e_Higher_and_Lower_Game.ipynb**: Build a simple higher/lower guessing game using LangGraph.

## Getting Started

1. Open any notebook in your preferred Jupyter environment (VS Code, Jupyter Lab, Jupyter Notebook, etc.).
2. Run the code cells and follow the instructions in each notebook.
3. Progress through the notebooks in order, or jump to any topic of interest.

## Purpose

This repository is intended for beginners who want to get hands-on experience with LangGraph by working through practical, step-by-step examples. Each notebook builds on previous concepts, making it easy to learn at your own pace.
